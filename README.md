# 🚦 Smart City Traffic Forecasting

## 📌 Overview
This project focuses on predicting traffic patterns at four junctions in a smart city using Machine Learning techniques. The goal is to help in traffic management and infrastructure planning.

---

## 📊 Dataset
- **Train Data**: Historical traffic data with vehicle counts  
- **Test Data**: Used for predicting future traffic  

---

## 🛠️ Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Prophet (Time Series Forecasting)

---

## ⚙️ Project Workflow
1. Data Loading & Cleaning  
2. Feature Engineering (Hour, Day, Month, Weekend)  
3. Exploratory Data Analysis (EDA)  
4. Model Building using Prophet  
5. Traffic Forecasting  
6. Prediction on Test Data  

---

## 📈 Key Insights
- Traffic peaks during **morning (8–10 AM)** and **evening (5–8 PM)**  
- Weekends show **lower traffic volume**  
- Some junctions have consistently higher congestion  
- Clear daily and weekly traffic patterns observed  

---

## 📁 Files Included
- `Smart_City_Traffic_Forecasting.ipynb` → Full implementation  
- `traffic_predictions.csv` → Predicted traffic values  

---

## 🎯 Conclusion
This project demonstrates how time series forecasting can be used to improve urban traffic systems and support smart city development.
